# RuleSync

Ferramenta para automatizar a criação de Appearance Profiles no Navisworks a partir de Excel.

## Problema
Criar regras visuais no Navisworks manualmente é lento, repetitivo e propenso a erro.

## Solução
O RuleSync converte planilhas estruturadas em XML compatível com Appearance Profiler.

## Stack
- Python
- CustomTkinter
- OpenPyXL

## Execução
```bash
pip install -r requirements.txt
python gui.py
